Capitalize Challenge

Instructions

Take the variable myString and capitalize the frist letter of the word using some of the methods that we talked about in the last video. Put the result in a variable called myNewString

Expected Result

        
            const myString = 'developer';
            console.log(myNewString); // 'Developer'

Hints

  1. You can use the charAr() method as well as string[index] to get the character at a specific index.
  2. The .toUpperCase() method will make the entire string uppercase.
  3. substring() or slice() will return a specific portion of a string.
