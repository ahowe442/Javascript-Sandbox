Data Types

Primative Data Types

  • typeof "John Doe" // Returns "string"
  • typeof 3.14 // Returns "number"
  • typeof true // Returns "boolean"
  • typeof 234567890123456789012345678901234567890n // Returns bigint
  • typeof undefined // Returns "undefined"
  • typeof null // Returns "object" (kind of a bug in JavaScript)
  • typeof Symbol('symbol') // Returns Symbol

Reference Types

These are dynamic in nature.

  • Object literals
  • arrays
  • functions

Static Types vs. Dynamic Types

Javascript is Dynamic, meaning that we don't declare a type.


TypeScript

This is a subset of Javascript that is statically typed, meaning it declares the type. The benefit is that it's less error prone.
